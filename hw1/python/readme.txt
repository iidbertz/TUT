Explanation of files:

main.py:
Main program provided by the course. Reads input, parses commands and calls the class
written by students. Run this with "python3 main.py". DO NOT EDIT main.py!!

datastructure_stub.py:
Empty template for students' assignments. Copy this to file datastructure.py and
start coding!

datastructure.py:
Implementation of the actual programming assignment, written by students. Implement the
given methods, add your own stuff as you like.
