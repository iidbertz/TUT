Projektin juurikansiossa (ylimmän tason kansiossa) on valmiiksi tehty
tiedosto .gitignore, jossa on listattu, minkä tyyppisiä tiedostoja ei
ole järkevää tallentaa versionhallintaan.  Ainakaan tällä opintojaksolla.

Periaatteessa tiedoston sisällöstä ei ole tarpeen välittää, mutta
voit halutessasi tutkia sitä.  Älä kuitenkaan muokkaa sen sisältöä.

Kun olet lukenut tämän tiedoston, voit halutessasi poistaa sen komennolla:

    git rm README.txt
